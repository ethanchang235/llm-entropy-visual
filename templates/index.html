LLM Entropy Visualizer - Help

This tool visualizes the next token predictions of a Language Model (LLM) and calculates the Shannon entropy of its probability distribution.

Features:

  • Text Input: Type any text. Predictions update automatically.
    • Ctrl+Enter or Cmd+Enter to submit immediately.
    • Esc to clear the input field.
  • Top K Tokens: Adjust how many of the most probable next tokens are displayed.
  • Temperature: Controls the randomness of the predictions. Lower is more deterministic, higher is more random.
  • Top-P (Nucleus Sampling): Filters predictions to the smallest set of tokens whose cumulative probability exceeds 'P'. A P of 1.0 is off. Lower values make output more focused.
  • Entropy: Measures uncertainty. Low = confident, High = uncertain.
  • Top K Cumulative Probability: Sum of probabilities for the displayed Top K tokens.
  • Input Tokenization: Shows how your input text is tokenized.
  • Probability Chart: Visualizes next token probabilities. Click a bar to append the (cleaned) token.
  • Log Scale for Probabilities: Toggle to view y-axis on a logarithmic scale. Useful for skewed distributions.
  • Show Raw Tokens in Chart: Toggle to display raw model tokens (e.g., with `ƒ†`) or cleaned tokens on the chart's X-axis.
  • Copy Top K: Copies tokens and probabilities to clipboard.
  • Surprise Me: Fills input with a random phrase.
  • Clear Button (Text Input): Clears the text input area (also Esc key).
  • Input History: Recent inputs and their entropy. Click to reload. Can be cleared.
  • Entropy Trend: Plots entropy against input length for the current session. Can be cleared.
  • Persistent Settings: Top-K, Temperature, Top-P, Log Scale, and Raw Token display preferences are saved in your browser.

Model Used: GPT-2 (Small)
